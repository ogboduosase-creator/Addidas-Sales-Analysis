# Adidas Sales Analysis (Excel)

## Project Overview
This project is an Excel-based sales analysis using the Adidas retail sales dataset. 
The purpose of the project is to analyze sales performance, profitability, and trends 
across different regions, retailers, products, and sales methods.

The project demonstrates my ability to clean data, perform analysis, and present 
business insights using Microsoft Excel.

---

## Dataset Description
The dataset contains retail sales information, including:
- Retailer and Retailer ID
- Region, State, and City
- Product category
- Sales method (In-store, Online, Outlet)
- Units sold
- Price per unit
- Total sales
- Cost of goods sold
- Operating profit
- Operating margin

---

## What I Did
- Cleaned and prepared the raw dataset for analysis
- Created calculated fields and key metrics
- Built pivot tables to summarize sales performance
- Designed charts to visualize trends
- Developed an interactive Excel dashboard with slicers

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

---

## Key Insights
- Certain regions generated higher total sales compared to others.
- Some products recorded high sales volume but lower profit margins.
- Sales method had a noticeable impact on revenue and profitability.
- A few retailers contributed a significant portion of total sales.
